# morose

Run npm registry & proxy on your own servers.

## Installation

```sh
$ npm install @bleenco/morose -g
```

## Usage

### Running a Server

```sh
morose
```

### Setting up npm config to use you morose

```sh
npm set registry http://localhost:10000
```
